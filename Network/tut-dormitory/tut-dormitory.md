# tut-dormitory config
豊橋技科大学生宿舎用の設定ファイルです。
設定情報は以下の通りです。
```
ユーザー名: admin
パスワード: admin

ポートについて:
FE0/0 (一番左): WAN (壁の情報コンセントと接続)
FE0/1 (中央): 使用しない
FE1/0 (右側の4つのポート): LAN (Wi-Fi APやPCを接続)

ルーター自身のIPアドレス: 192.168.1.254/24

DHCP: 
    範囲: 192.168.1.101 - 192.168.1.200
    デフォルトゲートウェイ: 192.168.1.254
    DNSサーバー: 192.168.1.254
    リース時間: 3600秒

NTP: インターネットマルチフィードのサーバーを使用
    NTP1: 2001:3a0:0:2001::27:123
    NTP2: 2001:3a0:0:2001::57:123
    NTP3: 2001:3a0:0:2001::87:123

DNS: CloudflareのDNSを使用
    DNS1: 1.1.1.1
    DNS2: 1.0.0.1

transix:
    ゲートアドレス: 2404:8e01::feed:100

管理:
    http-server: 有効
    telnet-server: 有効
```
